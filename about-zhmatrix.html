The Return — 14 Years Later

The End of an Era

The original zhplugin was a legendary mod for Counter-Strike: Source — but it could not survive forever. A major engine update to CS:S around 2012 broke compatibility with the plugin on a fundamental level. There was no patch, no fix — the plugin simply stopped working, and a complete rewrite would have been required to bring it back.

That was 14 years ago.

When zhplugin broke after the 2012 CS:S update, there was simply no comparable mod to replace it. No other plugin captured the same gameplay, the same feel. The server went offline — and stayed offline.

The Reborn

Now, 14 years later, ZOMBIEHORDE MATRIX - by Ħoζŷ.şħįŧ is back.

The Zombiehorde Matrix server is a complete rework of the original zhplugin concept — rebuilt from the ground up using modern SourceMod, with the original Zombie models, a weighted spawn system, RPG progression, dark atmosphere and the Matrix plugin. Everything that made the original great, brought back — and taken further.

  • Full zombie bot system with Ghost, Corpse, Zombie & Charple models
  • Headshot-Only mode — just like the original
  • RPG system with ranks, upgrades and player stats
  • Zombie respawn at death position — the horde never stops
  • Dark map atmosphere, fog, custom sounds & skins
  • Ghost zombie transparency, knife effects, bullettime
